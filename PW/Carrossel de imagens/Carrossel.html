Carrossel de Imagens

Navegue pelas imagens usando os botões, indicadores ou deixe o carrossel no modo automático

5s

Como usar o carrossel:

  • Use os botões "Anterior" e "Próxima" para navegação manual
  • Clique nos indicadores abaixo das imagens para ir direto a uma imagem específica
  • Use o botão "Pausar Auto" para interromper a navegação automática
  • Ajuste a velocidade do carrossel automático usando o controle deslizante
  • Passe o mouse sobre uma imagem para ver sua descrição
  • O carrossel é totalmente responsivo e se adapta a qualquer tamanho de tela
